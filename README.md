# smart-task-manager
Smart Task Manager with filtering, drag-and-drop, dark mode, and local persistence.

A clean and interactive task management web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add, edit, and delete tasks
- Task categorization (Personal / Work)
- Filter tasks (All / Completed / Pending)
- Search functionality
- Drag-and-drop task reordering
- Dark mode toggle
- LocalStorage persistence

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 💡 Future Improvements
- Backend integration (Node.js / Java)
- User authentication
- Cloud database
